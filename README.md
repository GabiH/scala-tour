Scala Tour
==========

Contains a Scala compiler and some impress.js presentations


## Compile And Run
 ```
Linux/Mac:
./sbt/sbt stage
./target/start

Windows can only compile
sbt\sbt stage
 ```
view
http://localhost:9000/p1/#/welcome
